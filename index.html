CSS attributes

Basic text attributes

Name Valeur Description
font-family serif, sans-serif, monospace, cursive, ... determine the style of the writing
font-size xxx px determine the size of the writing
font-weight bold, normal, light, 500, 600, ... determine the thickness of the writing
text-align left, center, right, justify determine the alignment of the text
color #XXX, #XXXXXX, rgb(x, x, x), blue, ... determine the color of the writing
background-color #XXX, #XXXXXX, rgb(x, x, x), blue, ... determine the color of the background
opacity 0.4, ... determiane the opacity/transparence of an element
background-image url(...) add an image as the background of the element
text-transform uppercase, lowercase, ... transform the text according a special feature
line-height 1.4 or xxx % or xxx px Determine the height of text lines
letter-spacing xxx px space between letters
word-spacing xxx px space between words

The box model

Name Valeur Description
width xxx px, xxx % determine the width of the box
height xxx px, xxx % determine the height of the box
padding xxx px determine the space interne of the box between the border and the content.
Can be top, right, bottom or left.
All value can be set up in one line
border xxx px, solid, ...., green, .... define the thickness, the form and the color of the border
margin xxx px determine the space externe of the box.
Can be top, right, bottom or left.
All value can be set up in one line
overflow scroll, determine how the display will react if the child exceed the parent size
border-radius 0.5 adjust corner of the box
box-sizing content-box or border-box set up the way the browser calculate the dimension of the box model
